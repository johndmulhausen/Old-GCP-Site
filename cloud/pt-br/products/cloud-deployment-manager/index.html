Cloud Deployment Manager

Cloud Deployment Manager allows developers to easily design, share, deploy and manage complex Cloud Platform solutions using simple, declerative templates. From simple web servers to complex highly available clusters, Deployment Manager allows teams to spend less time managing, and more time building.

Implantações simples e repetitivas na Cloud

O uso de modelos declarativos e parametrizáveis do Deployment Manager simplifica as operações na Cloud Platform e as torna repetitíveis, além de retirar das equipes de operação a sobrecarga do gerenciamento do estado da infraestrutura ou do gerenciamento de mudanças.

Deixe o Google cuidar da sua infraestrutura

O Deployment Manager pode monitorar ativamente o estado dos seus recursos durante e após a implantação do modelo. Ele pode considerar muitos tipos de recursos da Cloud Platform, eliminando a necessidade de conectar-se a muitas APIs.

Ferramentas diretas e integração profunda

Use nossa interface de linha de comando simples e programável para gerenciar suas implantações ou utilize uma API RESTful repleta de recursos, em conformidade com o protocolo OAuth 2.0, para uma integração completa com seu conjunto de ferramentas e fluxos de trabalho.
