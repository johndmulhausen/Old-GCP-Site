Cloud Deployment Manager

Cloud Deployment Manager allows developers to easily design, share, deploy and manage complex Cloud Platform solutions using simple, declerative templates. From simple web servers to complex highly available clusters, Deployment Manager allows teams to spend less time managing, and more time building.

Funciones

Implementaciones sencillas y repetibles en la nube

Gracias al uso de las plantillas declarativas y parametrizables del Gestor de implementación, las operaciones en Cloud Platform son más sencillas y se pueden repetir. Además, libera a los equipos de operaciones de la carga que supone gestionar el estado de su infraestructura o gestionar los cambios.

Dejar que Google cuide de la infraestructura

El Gestor de implementación puede supervisar activamente el estado de tus recursos tanto durante como después de la implementación de una plantilla. Puede hacerse cargo de muchos tipos de recursos de Cloud Platform, de modo que elimina la necesidad de conectar muchas API.

Herramientas sencillas e integración profunda

Utiliza nuestra sencilla interfaz de línea de comandos que permite añadir nuevas secuencias de comandos para administrar tus implementaciones o para sacar provecho de una API RESTful con todas las funciones y compatible con OAuth 2.0 para integrarla de forma profunda en tu propia cadena de herramientas y tus flujos de trabajo.
