Google Cloud Platform で
モバイル アプリを作成

モバイル アプリのバックエンドを作成し、ホストできます。インフラストラクチャが自動で管理されるため、アプリに集中することができます。さらに、何百万人ものユーザーをサポートできるよう無限に拡張できます。

今すぐ試すまたはお問い合わせ

300 秒でアプリをデプロイ

Google Cloud Endpoints でカスタム コードを使用してモバイル バックエンドを簡単に開発できます。自動生成される厳格に型指定されたクライアント ライブラリ（iOS、Android、JavaScript）からバックエンド API を簡単に呼び出すことができます。

Google Cloud Platform でモバイル アプリを作成するメリット:

自動スケーリングと負荷分散

アプリケーション コードを作成したら、後は Google にお任せください。どんなにユーザー数が多くても、アプリケーションが保存するデータが多くても、App Engine は必要に応じて拡張できます。

静的コンテンツ サービス

App Engine では、アプリケーション サーバーとは別の専用のサーバーとキャッシュから静的ファイルを表示しています。

データ処理

アプリケーション ログによってアプリの使用状況やバックエンドの動作が分析できます。BigQuery を使用すると、ログ ファイルのリアルタイムなアドホック分析情報を収集し、数秒でテラバイト サイズのデータを分析できます。

地理的近接の検索

App Engine の Search API を利用して地理的な近接を検索できます。

プッシュ通知

Android ユーザーには Google クラウド メッセージングを使用してデータを送信でき、iOS ユーザーには Sockets API を使用してプッシュ通知を送信できます。

データ ストレージ

Google Cloud Storage では、シンプルな REST ベースのインターフェースを使用してオブジェクトとファイルをテラバイト サイズまで保存できます。

スタート ガイド

モバイル アプリをデプロイするのに、コード、チュートリアル、Google Cloud Platform 以外必要ありません。
