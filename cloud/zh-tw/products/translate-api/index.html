Translate API

利用 Google 翻譯，快速翻譯上千種語言的應用程式。

立即試用

功能

靈活運用語言

Google Translate API 可以翻譯多種語言。只要利用 RESTful API 即可偵測文件的語言，然後翻譯成其他語言。

透過 Google API 進行存取

標準的 Google API 提供 Google Translate API。請參閱程式碼範例和程式庫，瞭解 10 種不同的程式設計語言，包括 Python、Objective C 和 Ruby。

價格合理親民

Google Translate API 是按每傳送 1 百萬位的字元收價。詳情請參閱我們的價目表。

價格

Google Translate API 是按照使用量計費，翻譯使用量的計算方式採用百萬個字元數 (M) 為計算單位，其中 1 M = 106 個字元。詳情請參閱價格常見問題。

注意：您必須提供所有帳戶資訊，才可使用 Google Translate API 第 2 版。請參閱下列步驟，瞭解如何啟用計費功能。

  • 使用費用：
    • 翻譯：
      • 每 1 百萬個字元 $20；費用會視實際提供的字元數量比例而有所調整。舉例來說，假設您翻譯 50 萬個字元，Google 會向您收取 $10 的費用。
    • 語言偵測：
      • 每 1 百萬個字元 $20；費用視實際提供的字元數量比例而有所調整。
  • 使用限制：
    • Google Translate API 預設的使用上限為每天 2 百萬個字元。您可以參閱以下步驟，前往 Google Cloud Console，將這個限制放寬成每天 5 千萬個字元。
    • 如果您每天須翻譯超過 5 千萬個字元，請與我們聯絡

如要為您的專案啟用計費功能，請執行以下步驟：

  1. 前往 Google Cloud Console。
  2. 選取所需專案。
  3. 在左側導覽列，選取 [帳單]。
  4. 在網頁上方，選取 [啟用計費功能]。
  5. 填寫表格。

如要查看或變更專案的使用量限制，或者想放寬使用量限制，請進行以下步驟：

  1. 前往 Google Cloud Console。
  2. 選取所需專案。
  3. 在左側的導覽列中，選取 [API 與驗證]，然後選取所需的 API。
  4. 前往 API 的資訊網頁，選取 API 名稱旁的 [配額]。
