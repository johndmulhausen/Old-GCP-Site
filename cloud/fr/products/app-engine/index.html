App Engine

Exécutez vos applications sur une plate-forme en tant que service (PaaS) entièrement gérée. Profitez de services intégrés qui augmentent votre productivité. Pour commencer immédiatement à développer votre application, il vous suffit de télécharger le SDK.

Fonctionnalités

Langages et frameworks courants

Développez des applications dans certains des langages de programmation les plus courants, à savoir Python, Java, PHP et Go. Utilisez des frameworks que vous connaissez déjà, tels que Django, Flask, Spring et webapp2. Développez l'application en local grâce aux SDK propres à chaque langage. Associez vos applications à Compute Engine pour intégrer d'autres technologies connues, telles que Node.js, C++, Scala, Hadoop, MongoDB, Redis et bien d'autres.

Priorité à votre code

Nous nous occupons de l'administration des bases de données, de la configuration des serveurs, de la segmentation des données et de l'équilibrage des charges. La répartition du trafic vous permet de publier différentes versions de votre application pour effectuer des tests A/B. Vous pouvez également compartimenter les données de l'application grâce à une architecture mutualisée.

Nombreuses options de stockage

Choisissez le type de stockage dont vous avez besoin : une base de données MySQL traditionnelle avec Cloud SQL, un magasin de données NoSQL sans schéma ou un système de stockage d'objets avec Cloud Storage.

Services intégrés puissants

Avec App Engine, soyez plus productif et ne vous souciez plus du code récurrent. Développez l'application de votre choix en vous appuyant sur des services gérés tels que les files d'attente de tâches, Memcache et l'API Users.

Outils de développement connus

Utilisez des outils que vous connaissez déjà, comme Eclipse, IntellIJ, Maven, Git, Jenkins, PyCharm et d'autres. Le SDK d'App Engine vous permet de tester des applications en local dans un environnement de simulation, puis de les déployer à l'aide d'outils de ligne de commande simples d'utilisation ou du lanceur pour ordinateur de bureau.

Déploiement à l'échelle de Google

Certains des services Web les plus populaires au monde ont été développés sur notre plate-forme. Votre application évolue automatiquement pour accepter jusqu'à sept milliards de demandes par jour, mais aussi pour s'adapter aux baisses de trafic.

Sources d'information

  • Qu'est-ce que Google App Engine ?

    Google App Engine vous permet d'exécuter des applications Web au sein de l'infrastructure de Google. Les applications App Engine sont faciles à développer, à gérer et à adapter lorsque vos besoins en termes de trafic et de stockage des données augmentent. Avec App Engine, vous n'avez aucun serveur à gérer. Il vous suffit d'importer votre application pour que les utilisateurs puissent s'en servir.

  • Pourquoi utiliser App Engine ?

    Google App Engine vous permet de développer des applications Web à partir des mêmes systèmes évolutifs que ceux utilisés pour exécuter les applications Google. Les applications App Engine sont faciles à développer, à gérer et à adapter lorsque vos besoins en termes de trafic et de stockage des données augmentent.   • Fonctionnalités d'App Engine

    Cette page présente un récapitulatif des fonctionnalités d'App Engine. Certaines de ces fonctionnalités sont disponibles quel que soit l'environnement d'exécution, tandis que d'autres ne sont compatibles qu'avec certains langages. En général, les fonctionnalités se comportent de la même manière dans tous les environnements d'exécution compatibles, mais il existe des exceptions.
